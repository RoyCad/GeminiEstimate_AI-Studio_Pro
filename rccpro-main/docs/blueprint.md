# **App Name**: GeminiEstimate

## Core Features:

- PWA Compatibility: Allows users to install the application on mobile and desktop devices, offering offline capabilities via service workers.
- Responsive Layout: Adapts the layout to fit any screen size, ensuring a seamless experience across devices.
- User Authentication: Firebase Authentication integration for user registration and login using email/password, Google, or phone number.
- Project Management: Enables users to create, view, and edit different types of projects with detailed descriptions.
- Image-Based Estimation: Tool where users upload an image of a room, and AI (computer vision) estimates wall area and dimensions to determine paint or tile requirements.
- Real-time Market Analysis: Tool where the application fetches and displays current market prices for construction materials using the Gemini API, displayed in a dashboard.
- Report Generation: Generates professional PDF reports of complete project estimations, with options to download and share.

## Style Guidelines:

- Background color: Light gray (#F0F2F5) to provide a clean and modern look.
- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, contrasting well with the light background.
- Accent color: Light purple (#7986CB), for interactive elements and highlights, creating a cohesive and calming aesthetic.
- Body and headline font: 'PT Sans', a versatile and readable sans-serif, for clear and accessible text throughout the app.
- Use minimalist, clear icons to represent different project categories and actions.
- Card-based layout with subtle shadows and frosted glass effect on panels to give a modern look, as described by the user.
- Smooth transitions and animations on button clicks and data loading for a fluid, app-like experience.
